# Template de E-commerce com Checkout via WhatsApp

Este documento registra o escopo do template comercial usado para criar lojas online com vitrine, carrinho, painel administrativo e fechamento de pedido pelo WhatsApp.

## Objetivo

- Entregar uma loja pronta para pequenos negocios venderem com atendimento assistido.
- Manter um painel simples para produtos, categorias, cupons, estoque, clientes, pedidos e configuracoes da loja.
- Permitir identidade leve por cliente: nome, dominio, WhatsApp, Instagram, cidade, cores, imagens, SEO e politicas.

## Escopo do Template

- Home com vitrine e produtos destacados.
- Catalogo com busca, categorias e ordenacao.
- Pagina de produto com galeria, variantes e chamada para WhatsApp.
- Carrinho com cadastro do cliente e geracao de pedido.
- Admin protegido por Supabase Auth e allowlist de emails.
- Supabase Postgres por cliente.
- Supabase Storage para midias de produto.
- Deploy Netlify por cliente.

## Fora do Escopo Inicial

- Gateway de pagamento.
- Assinatura recorrente.
- Multi-tenant em um unico banco.
- Personalizacao visual profunda por cliente.
